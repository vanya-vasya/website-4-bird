# 🔄 Перезапуск сервера после изменений

## Проблема

После изменений в коде видишь старое поведение:
- Фейковые токены: `test_1760002870784_84xzzpna0`
- Старые ошибки
- Изменения не применяются

## ✅ Решение

### Локальный сервер (npm run dev)

1. **Останови сервер:**
   - Нажми `Ctrl+C` в терминале где запущен сервер
   - Или закрой терминал

2. **Очисти кеш Next.js (опционально):**
   ```bash
   rm -rf .next
   ```

3. **Запусти сервер заново:**
   ```bash
   npm run dev
   ```

4. **Очисти кеш браузера:**
   - Chrome/Edge: `Ctrl+Shift+R` (Windows) или `Cmd+Shift+R` (Mac)
   - Или Hard Refresh в DevTools

### Проверка

После перезапуска создай новый платеж и проверь логи:

**✅ Правильно (должно быть):**
```
Token: 15e9c004587bc3f9b8e789041cb502da7679f7a18f6cb7ca7b1b0226a527a8a8
                 ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
                 64 символа (hex) - настоящий токен от NetworxPay
```

**❌ Неправильно (старый код):**
```
Token: test_1760002870784_84xzzpna0
       ^^^^                         
       Фейковый токен
```
